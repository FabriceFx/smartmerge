🚀 Guide de Démarrage : SmartMerge

📋 1. Préparer vos données

L'outil peut générer une feuille d'exemple pour vous. Allez dans Extensions > SmartMerge > 🧪 Générer données de test. Cela créera un onglet avec des contacts prêts à l'emploi. La première ligne utilisera automatiquement votre e-mail de compte Google pour tester !

🛠️ 2. Lancer l'application

Assurez-vous d'être sur l'onglet contenant vos contacts, puis cliquez sur SmartMerge > 🚀 Tableau de bord pour afficher vos statistiques.

✉️ 3. Rédiger votre campagne

  • Cliquez sur Nouvelle campagne.
  • Vérifiez la Colonne e-mail.
  • Cliquez sur les étiquettes bleues (ex: {{Prénom}}) pour insérer des variables personnalisées dans l'objet ou le message.
  • L'aperçu en direct (à droite) vous montre exactement le résultat final.

🎯 4. Gérer les destinataires et les modèles

À droite, décochez simplement les personnes à qui vous ne souhaitez pas envoyer le message. Vous n'avez pas besoin de les supprimer de votre tableur.

Si vous comptez réutiliser ce mail, cliquez sur Sauvegarder le modèle. L'objet, le texte, le nom d'expéditeur et les options seront mémorisés pour la prochaine fois !

📊 5. Suivi des Ouvertures (Tracking)

Dans la fenêtre de rédaction, vous pouvez cocher "Suivi des ouvertures". Un pixel invisible sera ajouté à chaque e-mail pour savoir quand il est lu.

⚠️ Comment activer le tracking (À FAIRE UNE SEULE FOIS) Pour que le pixel fonctionne, vous devez "publier" ce script sur le web afin qu'il puisse être contacté :
  1. Allez dans le menu Extensions > Apps Script.
  2. En haut à droite, cliquez sur Déployer > Nouveau déploiement.
  3. Sur la roue crantée (⚙️), choisissez : Application Web.
  4. Exécuter en tant que : Moi.
  5. Qui a accès : Tous (Très important ! Sinon le tracking ne marchera pas).
  6. Cliquez sur Déployer.

Une fois cette opération effectuée, la colonne "Date d'ouverture" se remplira automatiquement au fil de l'eau, et votre tableau de bord affichera votre taux de réussite !

⚖️ 6. Conformité RGPD et Anti-Spam

Règles d'or pour atterrir en boîte de réception
  • B2B vs B2C : En B2C (particuliers), le consentement préalable (opt-in) est obligatoire. En B2B (professionnels), vous pouvez envoyer un e-mail sans consentement si l'offre est liée à leur profession, mais l'opt-out reste obligatoire.
  • Authentification du domaine : Il est aujourd'hui indispensable de configurer les protocoles SPF, DKIM et DMARC sur votre nom de domaine via votre hébergeur web ou la console Google Workspace.
  • Désinscription : L'outil intègre une option "Lien de désinscription automatique". Utilisez-la systématiquement pour rester en conformité avec la loi.
  • Hygiène de la base : Supprimez les adresses qui n'ouvrent jamais ou qui n'existent plus. L'outil vous y aide grâce à la colonne "Statut SmartMerge".
